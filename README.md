# Customer Churn Prediction

A Machine Learning project that predicts whether a customer is likely to leave a company (customer churn). The project compares multiple classification models, evaluates their performance, and explains predictions using SHAP for better business decision-making.

---

## Features

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature engineering
- Customer churn prediction using multiple ML models
- Model comparison and evaluation
- Explainable AI with SHAP
- Business insights for customer retention

---
